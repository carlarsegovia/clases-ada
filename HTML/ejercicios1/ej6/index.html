Curso de desarrollo Frontend

En este curso obtendrás conocimientos claves para iniciarte en el mundo de la programación Frontend. Aprenderás a utilizar un entorno de desarrollo integrado (IDE) para escribir tu código con el que usarás el lenguaje HTML para crear la estructura de tu página, conocerás el lenguaje CSS para personalizar su apariencia y te enseñaremos cómo funciona el lenguaje JavaScript para agregarle interactividad y hacerlo dinámico.

  1. Introducción a HTML.
  2. Introducción a CSS.
  3. Introducción a Sass.
  4. Introducción a JavaScript.
  5. Introducción a React.
